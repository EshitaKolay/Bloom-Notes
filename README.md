# Bloom Notes
Bloom Notes is a simple and intuitive note-taking application built with HTML, CSS, and JavaScript. It allows users to organize notes into folders, create and edit notes, and manage their content efficiently through a clean and user-friendly interface.
## Features

* Create, edit, and delete notes
* Organize notes using custom folders
* Automatic local storage saving
* Simple and responsive design
* Quick note navigation
* Lightweight and fast performance

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Local Storage API

This project was developed to practice front-end development concepts, including DOM manipulation, event handling, data persistence, and user interface design.
